A premium, world-class trade platform designed for **China Economic Mall**, Ghana's leading destination for quality wholesale and retail commerce. This application bridges the gap between international trade and the local Ghanaian market with a sophisticated digital experience.

---

## ✨ Design Philosophy: "Gold & Obsidian"
The platform utilizes a signature **Dark & Gold** aesthetic designed to evoke trust, luxury, and the high-value nature of global trade.
*   **Typography**: `Playfair Display` (High-end Serif) for headers & `DM Sans` (Modern Sans) for body.
*   **Colors**: Obsidian Black (`#0a0a0f`) and Brushed Gold (`#c9a84c`).
*   **Atmosphere**: Frosted glass effects, ambient radial glows, and smooth motion transitions.

---

## 🚀 Key Features
*   **Immersive Hero Section**: A high-impact entry point featuring trade statistics and an interactive mall spotlight.
*   **Intelligent Store Directory**: Searchable, categorized catalog of over 500+ stores across all mall levels.
*   **Advanced Loyalty System**: A gamified rewards tier (Bronze, Silver, Gold) with animated progress tracking for frequent shoppers.
*   **Real-time Promotions**: Dedicated section for mid-year mega sales and store-specific flash deals.
*   **Mobile-First Design**: A sophisticated mobile slide-down menu and responsive grid layouts for shopping on the go.

---

## 🛠️ Tech Stack
*   **Frontend**: React 19, TypeScript, Vite
*   **Styling**: Tailwind CSS v4 (Alpha/Beta engine)
*   **Animations**: Framer Motion (Page transitions and micro-interactions)
*   **Icons**: Lucide React & Phosphor Icons
*   **Backend**: Express.js (serving API endpoints)
*   **ORM**: Prisma with PostgreSQL support

---

## 📦 Getting Started

### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ko2527600/China-Economic-Mall-.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Set up Environment**:
    Create a `.env` file based on `.env.example`.

4.  **Run in Development**:
    ```bash
    npm run dev
    ```
